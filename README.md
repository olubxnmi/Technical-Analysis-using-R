## Overview

This repository folder contains separate R implementations for technical analysis indicators. The assignment requires using standard R functions only, without existing technical analysis libraries.

## How to Run

1. Open RStudio.
2. Set the working directory to this assignment folder.
3. Open `tests/run_tests.R`.
4. Run the full script.
5. Review the output for each indicator.
